مبتدی 1. تنظیم DHCP Server
DHCPNetwork
▾
📌 توضیح
توزیع خودکار IP، Gateway و DNS به کلاینت‌های شبکه Host-Only (محدوده ۱۹۲.۱۶۸.۱۳۷.۲ تا ۲۰۰).
💡 سرنخ ▸
نصب نقش DHCP، ایجاد Scope با محدوده‌ی ۱۹۲.۱۶۸.۱۳۷.۰/۲۴، تنظیم Gateway=۱۹۲.۱۶۸.۱۳۷.۱ و DNS=همان سرور.
📘 آموزش گام‌به‌گام ▸
۱. Server Manager → Add Roles → DHCP Server ۲. پس از نصب، DHCP Console را باز کنید. ۳. روی IPv4 راست کلیک → New Scope. ۴. نام: HostOnly-Scope، محدوده: 192.168.137.2 تا 192.168.137.200. ۵. Exclusion: (اختیاری) 192.168.137.1 را استثنا کنید. ۶. Lease Duration: 8 hours. ۷. Gateway: 192.168.137.1 ۸. DNS: 192.168.137.1 (سرور خودتان) ۹. فعال‌سازی Scope.
